## Travel Journal
- A React project displaying data using props
- from scrimba.com Frontend Path Course

# Requirements
- npm
- React
- Vite

# How to run
- have ```npm``` installed
- unpack the repository to your local storage
- ```npm create vite@latest```
- cd <project_path>
- ```npm install```
- ```npm run dev```
- go to ```localhost:5173``` in your browser
